KaseyBon
========

Building the new [kaseybon.com](http://kaseybon.com)

Install
-------

1. Install Grunt

  ```$ npm install -g grunt-cli```
1. Install Node Modules

  ```$ npm install```

Website compiles to ```build/``` folder
